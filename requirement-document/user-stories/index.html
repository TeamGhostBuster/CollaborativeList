User Story¶

  1. Authancation

  2. 1.1 As a user, I can login using my Google account

  3. Create

  4. 2.1 As a user, I can create a reading list.
  5. 2.2 As a user, I can create a reading group.
  6. 2.3 As a user, I can add metadata tags to each item.

  7. 2.4 As a user, I can create reading lists for the group.

  8. Edit

  9. 3.1 As a user, I can make note on each item.
  10. 3.2 As a user, I can comment on each item.
  11. 3.3 As a user, I can rename lists.
  12. 3.4 As a user, I can merge lists.
  13. 3.5 As a user, I can partite lists.
  14. 3.6 As a user, I can copy items from one list to another.
  15. 3.7 As a user, I can move items from one list to another.
  16. 3.8 As a user, I can delete items from lists.
  17. 3.9 As a user, I can archive lists.

  18. 3.10 As a user, I can retrived archived lists.

  19. Share

  20. 4.1 As a user, I can grant/revoke read/write access of reading list to others.
  21. 4.2 As a user, I can share my reading list to a group.
  22. 4.3 As a user, I can upvote/downvote each item once.

  23. 4.4 As a user, I can hide my comments on items.

  24. Admin

  25. 5.1 As an admin, I can invite people to join my group.
  26. 5.2 As an admin, I can block people in my group.
  27. 5.3 As an admin, I can kick out people from my group.
  28. 5.4 As an admin, I can delete an item and keep people from reposting it.
  29. 5.5 As an admin, I can set a limit on how many suggestions each person can submit

  30. 5.6 As an admin, I can make someone else in the group the admin.

  31. Search/Filter

  32. 6.1 As a user, I can search item by metadata (tags).
  33. 6.2 As a user, I can search item by content (keyword).
  34. 6.3 As a user, I can filter item by metadata (tags).
  35. 6.4 As a user, I can filter item by content (keyword).

  36. 6.5 As a user, I can search a group by group name (keyword).

  37. Notification

  38. 7.1 As a user, I can receive voting notification whenever it changes.
  39. 7.2 As a user, I can receive notification when the voting end.
